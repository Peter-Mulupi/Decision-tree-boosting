# Decision Tree and AdaBoost Project

## Description
This project implements a Decision Tree classifier and improves it using AdaBoost (Boosting technique). The model is trained on the Iris dataset.

## Features
- Decision Tree classification
- Model evaluation using accuracy and confusion matrix
- Tree visualization
- Boosting with AdaBoost

## Requirements
- scikit-learn
- matplotlib

## How to Run
```bash
pip install -r requirements.txt
python app.py
